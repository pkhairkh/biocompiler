# Evaluation & Benchmarking

This directory will contain benchmarking scripts, comparison baselines, and evaluation results.

## Planned Evaluation

- **Gene set**: β-globin (HBB), GFP, CFTR, BRCA1, and 6 additional genes
- **Baselines**: GeneDesign, DNAWorks, SpliceAI (as scoring comparators)
- **Metrics**:
  - Verdict distribution (PASS/FAIL/UNCERTAIN) across gene set
  - Certificate generation and verification time
  - Comparison with baseline tool predictions where applicable
